Project Overview

아이리버는 사운드 제품 정체성을 유지하면서 보다 많은 사람들이 쉽게 이용하고 즐길 수 있는 생활 속 전자제품을 판매하는 사이트입니다.
기존 웹사이트에는 단조로운 레이아웃과 정체성을 알 수 없는 모습을 보였습니다.
이를 보완하고자 유동적인 레이아웃을 추가하고 정체성을 확대할 수 있는 제품 이미지 위주로 웹 사이트를 리뉴얼 진행하였습니다.

기존 사이트 네비

As is

홈페이지의 정체성을 알 수 없음
사용성의 불편함
중복되는 네비
단조로운 효과

To be

대표하는 제품이미지 사용 → 어떤 홈페이지인지 직관적으로 확인
편리성 제공
중복되는 네비 제거 하나 로 통합
다양한 효과 추가

Goal

유동적인 레이아웃을 통해 소비자의 호기심을 느끼게 만들고
누구나 쉽게 이용 할 수 있도록 직관적이고 심플하게 레이아웃을 구성하는 것을 목표로 삼았습니다.

Keyword

Intuitive, Dynamic, Simple

직관적, 유동적, 깔끔함

소비자들의 시선을 사로잡을 수 있도록 유동적인 효과를 추가하고
오랜 기간 지속적으로 사용 할 수 있도록 깔끔하고 직관적으로 디자인하였습니다.

IA

TYPOGRAPHY

아이리버 제품 이미지의 깔끔함과 동일하게 깔끔하면서도 가독성이 높은 것이 특징인 Noto Sans 서체를 국문, 영문에 사용하였습니다.

COLOR

브랜드 공식 컬러를 사용하여 아이리버의 아이덴티티를 강조하였으며
컬러에 시선이 가지 않도록 최소한의 컬러를 사용하였고
제품을 강조 할 수 있도록 채도를 조절하여 색상과 최소한의 컬러로 구성하였습니다.

메인페이지

아이리버 대표 제품을 중앙에 배치하여
시각적으로 눈길이 가게 배치
좌측에 제품 설명을 추가함으로
제품 정보도 추가적으로 제공

카테고리별 어떤 제품들이 있는지
시각적으로 알아 볼 수 있도록 배치

다양한 아이리버의 소식을 짧은 시간에도
한눈에 다양한 소식을 볼 수 있도록
슬라이드 형식으로 배치

사람들이 일상 생활 속에서 아이리버를 사용하는
모습을 공유한 sns를 모아서 배치하여
아이리버가 일상 속에서 많이 쓰임을 알림

아이리버의 Youtube 홍보 영상을 통해
신제품 및 이벤트 제품에대한 관심도를 높임

아이리버를 사용한 사람들의 후기를 통해
상세페이지에 적힌 부분 외의 추가적인
정보를 얻을 수 있도록 제품 이미지와 함께 배치
